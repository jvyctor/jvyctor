# João Vyctor

Desenvolvedor Full Stack com foco em APIs, integrações de sistemas e aplicações web modernas.

## Sobre mim

Atuo construindo soluções que conectam backend, frontend e regras de negócio com foco em usabilidade, organização e entrega real. Meus projetos aqui no GitHub mostram principalmente aplicações web, APIs REST, dashboards e interfaces orientadas a produto.

## Stack principal

- TypeScript
- Next.js
- React
- .NET / ASP.NET Core
- NestJS
- Prisma
- PostgreSQL
- Redis
- Docker

## Projetos em destaque

### [API-TCE-CE](https://github.com/jvyctor/API-TCE-CE)

Proxy full stack para consulta dos dados abertos do TCE-CE, com API em ASP.NET Core e interface em Next.js para pesquisa dinâmica de recursos, filtros e municípios.

**Stack:** ASP.NET Core, Next.js, TypeScript, Docker

### [Kanban](https://github.com/jvyctor/Kanban)

Plataforma Kanban full stack com autenticação, permissões por membro, comentários, convites, recuperação de senha e atualização em tempo real.

**Stack:** Next.js, NestJS, Prisma, PostgreSQL, Redis, Socket.IO

### [Aplicacao-Dentista](https://github.com/jvyctor/Aplicacao-Dentista)

Landing page profissional para clínica odontológica com foco em conversão, prova social e agendamento via WhatsApp.

**Stack:** Next.js, TypeScript, Tailwind CSS, shadcn/ui

### [fluxus](https://github.com/jvyctor/fluxus)

Projeto em TypeScript voltado para experiência web e construção de interface.

**Stack:** TypeScript
